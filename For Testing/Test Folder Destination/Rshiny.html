Folder Exploder

This application allows you to unravel (explode) a folder and copied the unraveled folder or files to the destination folder

If resursive option is selected, all individual files in each subfolder will be copied

This app is also useful if you want to extra a particular file type from a bunch of folder. For example, if you enable the resursive option and enter .pdf in the advanced option below, then all the pdf files will be copied

Advanced Options

This section allows you to explode files with selected characteristics using AND Condition
